# Nexify Connect — Landing Page

The official marketing landing page for **Nexify Connect** — *Connect Smarter*.

A premium all-in-one social ecosystem: messaging, calls, communities, an AI
assistant, music, fitness and focus, in one app.

## Tech
Static, self-contained `index.html` (no build step). Brand-styled with the
Nexify design system (deep-navy base, cyan → blue → purple gradients,
glassmorphism, neon glows, Inter typeface).

## Hosting
Deployed via **GitHub Pages** from the `main` branch root.

## Local preview
Open `index.html` directly in a browser, or serve the folder:

```bash
npx serve .
```
